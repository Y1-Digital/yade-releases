# Y.A.D.E. Releases

This repository hosts release artifacts and the Sparkle appcast feed for
[Y.A.D.E.](https://github.com/Y1-Digital/yade) — a macOS dictation app.

- **Appcast:** [`appcast.xml`](https://y1-digital.github.io/yade-releases/appcast.xml)
- **Binaries:** See the [Releases](https://github.com/Y1-Digital/yade-releases/releases) tab.

Nothing here is meant to be consumed by humans directly; the app auto-updates.
